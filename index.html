Test Analysis Report Generator

Either paste your Form Responses and Answer Key as CSV, or point the tool at a published Google Sheet link so it fetches the live data instead — pick whichever mode suits each section below.

Test Details

Form Responses

Paste the full "Form Responses" sheet as CSV (with header row). Question columns must be named "Question 1", "Question 2" etc. "Score" as "12 / 75" is parsed automatically; Attempts/Correct/Incorrect/Accuracy columns are used if present.
In the response Sheet: File → Share → Publish to web → select the "Form Responses 1" tab → format CSV → Publish. Paste that link here. Re-generating the report always pulls the latest submissions.

Answer Key

First column = question number, second column = correct letter(s) or value (comma-separated if more than one is accepted).
Same as above, but publish the "AnswerKey" tab instead. Once published, any edits to the key sheet reflect automatically next time you generate.

Score Bands (optional)

Comma-separated lower bounds, highest band first — e.g. "31,21,11,1" creates bands 31+, 21–30, 11–20, 1–10, and ≤0 (if any score is 0 or below). Leave blank to auto-generate 5 even bands from total marks.
